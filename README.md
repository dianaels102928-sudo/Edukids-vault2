✅ **Backend API** (8 Revenue Endpoints)
- `/health` - Health check
- `/ads.txt` - AdMob compliance
- `/api/v1/admob-config` - Get ad unit IDs
- `/api/v1/players/register` - Register player
- `/api/v1/license/check` - Check unlock status
- `/api/v1/ads/track` - Track ad impressions
- `/api/v1/sessions/start` - Start game session
- `/api/v1/analytics/revenue` - Revenue analytics
- `/privacy-policy` - Privacy policy page
- `/terms-conditions` - Terms & conditions

## 🚀 Quick Start

### 1. Clone Repository
```bash
git clone https://github.com/dianaels102928-sudo/Edukids-vault2
cd Edukids-vault2
```

### 2. Setup Backend
```bash
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
pip install -r requirements.txt
python backend/app.py
```

### 3. Test Endpoints
```bash
curl http://localhost:5000/health
curl http://localhost:5000/api/v1/analytics/revenue
curl http://localhost:5000/privacy-policy
```

### 4. Build APK (Automatic - FREE)
```bash
git push origin main
# APK builds automatically in Actions tab (~30 min)
# Download from: Actions > Latest Build > Artifacts
```

