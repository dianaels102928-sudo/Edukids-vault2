# Edukids Vault 2

## Overview
Educational Gaming Platform with AdMob & PayPal integration.

## Features
- Interactive educational games
- AdMob advertising integration
- PayPal payment processing
- User-friendly interface

## Installation

```bash
npm install
```

## Usage

```bash
npm start
```

## Technologies
- Node.js
- React
- AdMob SDK
- PayPal SDK
